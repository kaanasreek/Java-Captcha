# Java CAPTCHA Generator

This project is a simple CAPTCHA Generator developed using Java Swing.  
It generates a random CAPTCHA image with noise and allows users to verify the entered text through a user-friendly GUI.

## 🚀 Features
- Random CAPTCHA text generation
- CAPTCHA displayed as an image
- Noise lines added for security
- Refresh button to generate a new CAPTCHA
- Verification button to check user input
- Simple and clean graphical user interface

## 🛠 Technologies Used
- Java
- Swing (JFrame, JLabel, JButton, JTextField)
- Graphics & BufferedImage
- Random Class

## ▶ How to Run

1. Compile the program:
   javac CaptchaGUI.java

2. Run the program:
   java CaptchaGUI

## 📌 Purpose
This project was created as a beginner-level Java GUI project to understand:
- Event handling
- GUI components
- Graphics drawing
- Image generation in Java
